Notes API Documentation

Main Endpoint: https://notesapp-v4w2.onrender.com

GETGet All Notes

Endpoint: /api/notes/getAll

Authorization: Bearer Token required

Request:

GET /api/notes/getAll
Authorization: Bearer <your_token>

Response:

[
    {
        "_id": "note_id",
        "title": "Note Title",
        "content": "Note Content",
        "user": "user_id",
        "createdAt": "2024-08-18T12:00:00.000Z",
        "updatedAt": "2024-08-18T12:00:00.000Z"
    },
    ...
]

POSTCreate a New Note

Endpoint: /api/notes/create

Authorization: Bearer Token required

Request:

POST /api/notes/create
Content-Type: application/json
Authorization: Bearer <your_token>

{
    "title": "My Note",
    "content": "This is the content of my note."
}

Response:

{
    "_id": "new_note_id",
    "title": "My Note",
    "content": "This is the content of my note.",
    "user": "user_id",
    "createdAt": "2024-08-18T12:00:00.000Z",
    "updatedAt": "2024-08-18T12:00:00.000Z"
}

PUTUpdate a Note

Endpoint: /api/notes/update

Authorization: Bearer Token required

Request:

PUT /api/notes/update
Content-Type: application/json
Authorization: Bearer <your_token>

{
    "id": "your_note_id",
    "title": "Updated Title",
    "content": "Updated content"
}

Response:

{
    "_id": "updated_note_id",
    "title": "Updated Title",
    "content": "Updated content",
    "user": "user_id",
    "createdAt": "2024-08-18T12:00:00.000Z",
    "updatedAt": "2024-08-18T12:00:00.000Z"
}

DELETEDelete a Note

Endpoint: /api/notes/delete/:id

Authorization: Bearer Token required

Request:

DELETE /api/notes/delete/:id
Authorization: Bearer <your_token>

Response:

{
    "message": "Note deleted successfully"
}

POSTDelete Multiple Notes

Endpoint: /api/notes/deleteMany

Authorization: Bearer Token required

Request:

POST /api/notes/deleteMany
Content-Type: application/json
Authorization: Bearer <your_token>

{
    "ids": ["note_id1", "note_id2", "note_id3"]
}

Response:

{
    "message": "Notes deleted successfully"
}

Authentication API Documentation

POSTLogin

Endpoint: /api/auth/login

Request:

POST /api/auth/login
Content-Type: application/json

{
    "email": "user@example.com",
    "password": "userpassword"
}

Response:

{
    "token": "your_jwt_token"
}

POSTRegister

Endpoint: /api/auth/register

Request:

POST /api/auth/register
Content-Type: application/json

{
    "name": "Your Name",
    "email": "user@example.com",
    "password": "userpassword"
}

Response:

{
    "_id": "new_user_id",
    "name": "Your Name",
    "email": "user@example.com",
    "createdAt": "2024-08-18T12:00:00.000Z",
    "updatedAt": "2024-08-18T12:00:00.000Z"
}
